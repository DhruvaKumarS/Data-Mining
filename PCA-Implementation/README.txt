Part 1 - PCA
-Make sure the pca_script.py and the dataset file are in the same directory.
-Run the file using python2 as follows:
	python2 pca_script.py
-You will be prompted to enter the filename as follows:
	Enter filename: <Type the filename>
	e.g. Enter filename: pca_a.txt
-Three plots will be displayed. One each for PCA, SVD and t-SNE.
-Repeat the steps for other dataset files.
